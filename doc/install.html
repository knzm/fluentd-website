インストール¶

ユーザは次の4つの方法でFluentdをインストールすることができます。

  • バイナリパッケージ
  • RubyGems
  • .tar.gz
  • Gitリポジトリ

ほとんどの場合バイナリパッケージからのインストールをお勧めします。

バイナリパッケージからのインストール¶

Fluentdの安定板は Treasure Data, Inc. からtd-agentという名称で提供されています。もし、自分でRubyインタプリタ (> 1.9.2)をインストールしたくない場合、またFluentdとプラグインのバージョンを管理したくない場合にはtd-agentを使うのがお勧めです。

td-agentは以下のコンポーネントで構成されています:

  • Fluentd専用のRuby1.9.3インタプリタ(ノード上でRubyをインストールする必要はありません)
  • Fluentd
  • jemalloc (メモリの断片化を防ぎます)
  • 安定して互換性のあるバージョンのプラグイン

td-agentは商用環境で使われており、また活発にメンテナンスされています。td-agentのインストールについて、詳細は下記を参照してください:

  • DebianおよびUbuntuへのtd-agentインストール
  • RedhatおよびCentOSへのtd-agentインストール
  • td-agentのChangeLog

注意: td-agentでFluentdをインストールした場合、サーバー上のコマンドは’fluentd’ではなく’td-agent’になります。また設定ファイルは’/etc/td-agent/td-agent.conf’に置かれます。

RubyGemsからのインストール¶

RubyGemsでFluentdをインストールするためには、まず Ruby >= 1.9.2をインストールする必要があります。Rubyのインストールには rvm (Ruby Version Manager) をお勧めします。インストールされたRubyのバージョンは次のように確認できます:

$ ruby --version
ruby 1.9.2p180 (2011-02-18 revision 30909) [x86_64-darwin10.6.0]

Ruby >= 1.9.2 をインストール後、次の gem コマンドを実行します。

# system-wide ruby
$ sudo gem install fluentd

# via rvm
$ gem install fluentd

.tar.gzからのインストール¶

ソースパッケージをbuildする前に、以下のライブラリがインストール済みであることを確認して下さい:

  Ubuntu/Debianでのインストール RedHat/CentOSでのインストール
openssl apt-get install libssl-dev yum install libssl-dev
zlib apt-get install zlib1g-dev yum install zlib-devel
readline apt-get install libreadline6-dev yum install readline-devel

次に ダウンロードページ からソースパッケージをダウンロードし、展開してください。

最後に次のコマンドでbuildとインストールを行います:

$ ./configure
$ make
$ sudo make install

上のコマンドでは、Rubyが $prefix/lib/fluent/ruby に、Fluentdが $prefix にインストールされます。

最新Gitリポジトリからのインストール¶

最新の開発版をFluentdの ソースリポジトリ からインストールするためには、 Ruby >= 1.9.2 を用意し、次のコマンドを実行します:

$ git clone https://github.com/fluent/fluentd.git
$ cd fluentd
$ rake
$ gem install pkg/fluentd-*.gem

次のステップ: インストールの確認

インストールの確認¶

Fluentdが正常にインストールされたことを確認するには、次のコマンドを実行します:

$ fluentd --setup ./fluent
$ fluentd -c ./fluent/fluent.conf -vv &
$ echo '{"json":"message"}' | fluent-cat debug.test

最後のコマンドではFluentdに”debug.test”タグと’{“json”:”message”}’メッセージが送られ、インストールに成功していればFluentdは次のメッセージを出力します:

2011-07-10 16:49:50 +0900 debug.test: {"json":"message"}

アップデート¶

Fluentdがインストールされている場合、次のコマンドで最新バージョンにアップデートします:

$ sudo fluent-gem install fluentd
