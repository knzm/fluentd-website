Log everything in JSON

Get Started Plugins

Fluentd is a lightweight and flexible log collector. Fluentd receives logs as JSON streams, buffers them, and sends them to other systems like MongoDB, Hadoop, or even other instances of Fluentd.

Install Fluentd
RPM package (.rpm) Debian package (.deb) gem install fluentd
